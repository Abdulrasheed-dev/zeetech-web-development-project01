Key Features of Graphic design:

  • Page Layout: Arrange text, images, and graphics
    on a digital canvas to create a visually appealing page layout.

  • Typography: Select and manipulate fonts, sizes, and
    styles to convey meaning and create emphasis

  • Image Editing: Edit and enhance images using software
    tools like Adobe Photoshop.

  • Color Management: Control color consistency across
    different devices and media.
